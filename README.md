Scaffold for backend Dotnet 10 and frontend Angular 22 project

Backend
1. Install dotnet cli 10.0
2. dotnet run

Frontend
1. Install nvm
2. nvm install 26.3.0
3. nvm use 26
4. node -v && npm -v
5. npm install -g pnpm@11.5.2
6. pnpm -v
7. pnpm add -g @angular/cli@22.0.0
8. ng --version
9. cd frontend && pnpm i --frozen-lockfile
10. pnpm start
